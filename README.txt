
excel-testng: driving TestNG tests through MS Excel
===============================================================================

excel-testng is a small library that lets you provide TestNG input in Excel 
files. This is especially helpful in end to end automation testing and with 
this library, Excel files can be used to document the tests as well as execute
them using TestNG.

The project is hosted at Google Code: http://code.google.com/p/excel-testng/
